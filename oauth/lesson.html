简要描述：

  • 获取用户课程

请求URL：

  • /user/lesson

请求方式：

  • GET

参数：

参数名 必选 类型 说明
year 是 int 学年
semester 是 int 学期

注意：该参数在headers处传入

参数名 必选 类型 说明
Authorization 是 string 传参格式：Bearer ，中间用空格隔开，其中AccessToken为访问令牌
