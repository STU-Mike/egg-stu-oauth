# 介绍

stu-oauth 是汕头大学课程表团队开发的一个校园账号开放平台，它能够帮助使用户更安全地向第三方应用授权信息，而不需要在第三方应用中输入账号密码等敏感信息

# 特性

  • 基于Oauth2.0开放协议
  • 使用汕头大学校园网账号

# 为什么使用开放授权

随着课程表团队发展，我们的服务会越来越多，不仅仅局限于课程表，可能会存在多个应用，这些应用都需要进行用户账号认证；对于团队内部，我们同样需要开发各种系统，来提升我们的开发、运维、管理效率，这同样需要进行用户认证。因此，需要一套统一的账号系统去提升我们的开发效率，使我们的应用不必过分关注与登录部分，集中更多的精力去开发应用的核心功能。
